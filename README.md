## About the project
SpendingsTracker is a Spring Boot application designed to turn raw transaction data into actionable visual insights. It parses financial data from a local CSV source and generates a real-time dashboard.

*   **Total Overview:** Instantly see the sum of all expenses for the month.
*   **Category Breakdown:** A Doughnut chart showing which sectors (Housing, Groceries, Entertainment, etc.) are consuming the budget.
*   **Daily Trends:** A Bar chart mapping out spending habits day-by-day to identify peak spending periods.

### How to run:
1.  Compile and run the Spring Boot application.
2.  Open your browser and navigate to `http://localhost:8080`.
